# reduplication

experimental piece for 21W.764, Computational and Experimental Writing. built by Claude and I with <3: I did ideation + python script, Claude converted it into js, and we worked on modifications and feature implementation.

## Quickstart

```
npm install
npm run dev
```

Then open the URL Vite prints (usually http://localhost:5173).
